zsh-mkc [![Build Status](https://travis-ci.org/caarlos0/zsh-mkc.svg?branch=master)](https://travis-ci.org/caarlos0/zsh-mkc)
=======

zsh `mkdir && cd` plugin.

## Install

### Antibody

```sh
$ antibody bundle caarlos0/zsh-mkc
```

### Antigen

```sh
$ antigen bundle caarlos0/zsh-mkc
```

## Usage

```sh
$ mkc a/b/c/d/e/f/g
```
